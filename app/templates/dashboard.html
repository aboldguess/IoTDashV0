Device API Integration (Your Current Setup)

From your IoT devices on the same network, post sensor data to: http://192.168.56.1:8001/api/sensor/publish

Recommended authorization header: Authorization: Bearer {{ user.api_key }}

Alternative header: X-API-Key: {{ user.api_key }}

Legacy form field: include api_key={{ user.api_key }} in form data.

Door sensor values: send open/closed, on/off, or 1/0.

Example curl command (template for your devices) 
curl -X POST http://192.168.56.1:8001/api/sensor/publish \
  -H "Authorization: Bearer {{ user.api_key }}" \
  -F "topic=sensors/door/front" \
  -F "value=open"
